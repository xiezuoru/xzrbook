1.小学生人工智能入门

开发：徐晓刚、江涛（金华兰溪）

不会编写代码，能不能学习人工智能，能不能训练并推理模型？在浦育平台的创意积木的支持下，仅仅使用无代码工具也能学习人工智能。本工作坊介绍“之江汇”互联网学校的在线课程《人工智能入门课程》核心设计思路，介绍图像分类、语音分类、手势分类等任务的快速实现，重点演示如何不编写代码通过采集收据训练简单的人工智能模型，实现模型推理和部署。
